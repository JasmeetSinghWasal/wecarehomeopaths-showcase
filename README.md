## ✅ Features

- Health and wellness blog with CMS-managed content
- Google Reviews integration — live reviews displayed on site
- Contact form with email notifications
- WhatsApp chat widget for instant patient communication
- Fully responsive — mobile, tablet, desktop
- SEO optimized for local Toronto search
